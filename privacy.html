Statistics & tracking (Piwik)

This site uses Piwik, "…the only web analytics tool that provides full compliance with [European] data protection laws" to collect anonymous visitor information including operating system, browser, and (anonymized) IP addresses. This information is not personally identifiable.

Piwik is also configured to honour the "do not track preference" if set in your browser.

Piwik Cookies

If you don't opt-out, Piwik sets several cookies.

If you use the opt-out dialogue that appears the first time you come to this site, Piwik will still collect information, but it will not set any cookies in your browser.

Comments

This site uses the Disqus commenting system. You can read Disqus' privacy policy here. You can read about some criticisms of Disqus on the Wikipedia page.

If you use the opt-out dialogue that appears the first time you come to this site, disqus will be completely disabled on subsequent page loads.

SSL

This site uses SSL, which should protect data sent back and forth from casual eavesdropping.
